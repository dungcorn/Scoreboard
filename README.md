# Simple Scoreboard Minimal Android App

가장 단순한 형태의 Android WebView 프로젝트입니다.

## GitHub 업로드 순서
1. 새 GitHub 저장소 생성
2. 이 프로젝트 파일 전체 업로드
3. 기본 브랜치를 main으로 유지

## GitHub Secrets
- KEYSTORE_BASE64
- KEYSTORE_PASSWORD
- KEY_ALIAS
- KEY_PASSWORD

## 빌드
Actions > Build Signed APK > Run workflow

## 포함 기능
- 빨강/파랑 양쪽 터치 점수 증가
- 각 팀 되돌리기 버튼
- 전체 초기화
- 좌우 바꾸기
- 세로 고정
